impress.js


impress.js是一个供开发者使用表现层框架，它为你的ppt提供了舞台和场景，内容完全由自己发挥，只要你的css和js能力足够强大。


github地址：点这里

start

首先需要引入impress.js，大家都懂的。


在使用时，你需要一个div做为整个舞台的载体，让其 id="impress"，所有的场景都在这里显示。


你需要在script标签中敲入：impress().init() 来初始化impress。


你需要点击场景或按 ↑ → ↓ ← 来进行场景的切换。

可用的class

impress-not-supported : 检查你的浏览器是否支持impress.js，通常加在body上即可。


fallback-message : 浏览器不支持impress.js时显示的元素，可用于提示用户。


step : 创建一个场景，即一个新的幻灯片，这个非常重要，除非你不想让舞台超级炫酷。


一些API

impress().init() - 初始化演示；


impress().next() - 播放下一个幻灯片；


impress().prev() - 回到上一个幻灯片；


impress().goto( idx | id | element, [duration] ) - 通过指定的id或DOM元素切换场景；duration可以指定切换时间；id对于impress来说是可有可无的，因为他会为每一个场景设置一个"step-n"的id

重点来了

data-transition-duration="1000"


改变场景的切换速度，默认为1000

data-perspective="1000"

改变透视的深度，默认1000，如果设置为0（这是零哦），炫酷的3D效果将离你而去。

data-x="2825"

data-y="2325"

data-z="-3000"

设置场景相对于舞台中心的位置
